Juego de geografía donde tienes que adivinar a qué país pertenece cada bandera. Más de 160 países de todo el mundo, 3 niveles de dificultad y sistema de racha con puntos extra.

🔗 **[Jugar ahora → pedroiruegas.github.io/adivina-bandera](https://pedroiruegas.github.io/adivina-bandera/)**

---

## ¿Cómo se juega?

1. Elige una dificultad (fácil, normal o difícil)
2. Se muestra una bandera — selecciona el país correcto entre las opciones
3. Cada acierto suma puntos; si mantienes una racha, ganas puntos extra
4. Al final ves tu resultado con estadísticas de la partida

## Dificultades

| Modo    | Rondas | Opciones por pregunta |
|---------|--------|-----------------------|
| Fácil   | 10     | 3 opciones            |
| Normal  | 15     | 4 opciones            |
| Difícil | 20     | 5 opciones            |

## Características

- 🗺️ Más de 160 países de América, Europa, Asia, África y Oceanía
- 🏳️ Banderas reales via [flagcdn.com](https://flagcdn.com)
- 🔥 Sistema de racha con puntos bonus
- 🌙 Modo oscuro / claro con toggle
- 📱 Diseño responsivo para móvil y escritorio
- ⚡ Sin frameworks ni dependencias — HTML, CSS y JS puro

## Tecnologías

- HTML5 + CSS3 + JavaScript vanilla
- Banderas: [flagcdn.com](https://flagcdn.com) (CDN gratuito, ~250 países)
- Fuentes: [Syne](https://fonts.google.com/specimen/Syne) + [DM Sans](https://fonts.google.com/specimen/DM+Sans) via Google Fonts
